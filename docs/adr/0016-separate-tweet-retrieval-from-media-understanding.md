# Separate Tweet Retrieval From Media Understanding

In v3, the tweet retrieval service remains responsible for fetching the source tweet, replies, author data, metrics, and media references from one retrieval provider, while a separate media understanding service interprets the source tweet media for joke context gathering. We chose this boundary because media understanding requires different capabilities such as OCR, image reading, product UI interpretation, chart reading, and video frame analysis, and separating it lets the product improve media interpretation without turning tweet retrieval into a broad analysis service.

Source tweet media references are neutral retrieval pointers plus metadata, such as media kind, URL, preview URL, alt text, dimensions, or duration. They are not media bytes and they must not include interpretation such as whether the media is a chart, product UI, launch screenshot, or the main news-bearing element; those reads belong to media understanding and joke context gathering.

For the first v3 media understanding boundary, source tweets without media use text-only context, source tweets with images receive image reads for each image, and source tweets with videos receive a video read for one selected video. When there is more than one video, v3 selects only the longest available video; this is the sole selection rule, not a relevance claim. Skipped videos are an accepted v3 limitation. A video read is broader than an audio transcript: it may include transcript, sampled frame summaries, visible text, and explicit limitations when a live adapter cannot understand part of the video.

The Media Understanding Service remains provider-agnostic at the product boundary. For the first v3 live adapter, the implementation should prefer Vercel AI Gateway when it can satisfy the needed image reads and video reads, because it is lightweight and already fits the existing integration style; the accepted trade-off is some platform dependency while preserving provider-neutral saved-run and product contracts.
